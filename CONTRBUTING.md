The following is a set of guidelines for contributing to this project. These are just guidelines, not rules, so use your best judgment and feel free to propose changes to this document in a pull request.

## Code of Conduct

The project and everyone participating in it is governed by the [Interactive Particle Network Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code. Please report unacceptable behavior to me.

## How Can I Contribute?

- Reporting bugs
- Suggesting enhancements
- Your first code contribution
- Pull requests

Remember, contributions are not limited to code. We appreciate contributions to documentation, design, and more.

## Styleguides

### Git Commit Messages

- Use the present tense ("Add feature" not "Added feature")
- Use the imperative mood ("Move cursor to..." not "Moves cursor to...")
- Limit the first line to 72 characters or less
- Reference issues and pull requests liberally after the first line

### JavaScript Styleguide

All JavaScript must adhere to [JavaScript Standard Style](http://standardjs.com/).
